# Olist-Sales-Operations-Dashboard
This project is a comprehensive Power BI dashboard built using the Olist e-commerce dataset. The goal is to analyze sales, customer behavior, operational efficiency, and profitability to provide actionable business insights.

Key Features
-- Total Revenue, Order Count, and Average Order Value KPIs
-- Revenue trends over time with month-over-month and year-over-year growth
-- Top 10 products and top 10 states by revenue
-- Freight cost & delivery analysis, including profit margins and delays
-- Dynamic time intelligence measures: YTD revenue, MoM growth, YoY growth
-- Interactive visuals: line charts, stacked bars, pie charts, maps, KPIs
